採購 planning 的知識卡片把供應鏈與採購工作裡的高密度術語，拆成可獨立查閱的概念。採購社群的對話常一句話塞進三四個縮寫—「這顆料 NCNR，走 ship and debit，LT 八個月要 Risk Buy」—沒有共同術語表的人會卡在名詞，而錯過背後的判斷邏輯。

每張卡片只處理一個術語的核心概念、概念位置、可觀察訊號與判讀方式。卡片之間用相對連結互引，跟 [文章](/business/procurement-planning/) 互補：卡片是概念索引，文章是決策與情境推導。

## 備料與需求策略

備料策略卡片回答「一顆料該備多少、備多久、用什麼機制備」。這是 planning 的核心決策層。

| 卡片                                                               | 核心問題                       |
| ------------------------------------------------------------------ | ------------------------------ |
| [Lead Time](/business/procurement-planning/cards/lead-time/)       | 交期多長，決定要備多久的庫存   |
| [Forecast](/business/procurement-planning/cards/forecast/)         | 需求預測，是參考不是保證       |
| [Risk Buy](/business/procurement-planning/cards/risk-buy/)         | 缺料前先備關鍵料或原材料       |
| [Safety Stock](/business/procurement-planning/cards/safety-stock/) | 安全庫存，吸收需求與交期的波動 |
| [寄售](/business/procurement-planning/cards/consignment/)          | 供應商在你端備庫存，用了才付款 |
| [斷料](/business/procurement-planning/cards/stockout/)             | 產線沒料可用，採購的最高禁區   |

## 供應商與量產

供應商卡片回答「找誰買、換不換、怎麼把量分配出去」。

| 卡片                                                                 | 核心問題                         |
| -------------------------------------------------------------------- | -------------------------------- |
| [Second Source](/business/procurement-planning/cards/second-source/) | 第二三供應商，分散斷料與漲價風險 |
| [NPI](/business/procurement-planning/cards/npi/)                     | 新品導入，量產前的驗證爬坡       |
| [追料](/business/procurement-planning/cards/expediting/)             | 追料件在每個節點的進度，不是催船 |

## 料件規則與生命週期

規則卡片回答「這顆料受什麼商業規則約束、處在生命週期哪一段」。

| 卡片                                                                    | 核心問題                             |
| ----------------------------------------------------------------------- | ------------------------------------ |
| [Ship and Debit](/business/procurement-planning/cards/ship-and-debit/)  | 代理商價格保護機制，價格被原廠掌握   |
| [NCNR](/business/procurement-planning/cards/ncnr/)                      | 不可取消不可退，與改單截止日         |
| [料件生命週期](/business/procurement-planning/cards/lifecycle-status/)  | NRND / EOL / LTB，料件走向停產的訊號 |
| [不可能三角](/business/procurement-planning/cards/impossible-triangle/) | 交期、價格、品質無法三者兼得         |
