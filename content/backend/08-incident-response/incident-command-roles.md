## 判讀訊號

- IC / scribe / liaison 角色重疊或缺失
- 同一人兼太多角色、決策變 bottleneck
- decision boundary 不清、跨角色協商耗時
- handover 靠口頭交接、無書面 state
- 工程師被臨時 page 進事故、不知道角色與職責
