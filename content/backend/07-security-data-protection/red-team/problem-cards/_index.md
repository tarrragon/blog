本章的責任是把常見業務流程拆成單一問題卡。每張卡片只處理一個流程節點，主體是失效樣式與判讀訊號，案例作為觸發式證據參考。

## 使用方式

1. 先選一個流程問題卡。
2. 再比對判讀訊號與失效樣式。
3. 觸發條件成立時再引用案例做證據比對。
4. 最後交接到 7.2 / 7.3 / 7.4 / 7.6 / 7.7 或 8.x workflow。
