這個資料夾收錄**工程之外學到的小知識**（TIL、Today I Learned）— 單字字源、概念由來、歷史冷知識、科學概念都算。一篇只說一個知識點、寫下「當初是怎麼理解的」、之後忘了可以回來查。

子分類：

- [單字](vocab/)：學到的英文單字 — 字源拆解、由來故事、同字根家族

還沒成氣候的知識點直接放本資料夾根層、同主題累積三篇以上再開子分類。

---

## 跟其他資料夾的邊界

寫作前先依序評估：

1. **是 blog 本身設定 / Hugo / Markdown 議題？** → `posts/`
2. **是工作中用某工具遇到的事件？** → `work-log/`
3. **是中性 frame 的工作方法論？** → `record/`
4. **是從 case 抽出的工程原則？** → `report/`
5. **是 OS / 工具小技巧？** → `other/`
6. **跟工程無關、純粹是學到的知識點** → **本資料夾**

判斷軸是「跟工程工作的距離」：前五個資料夾都服務工程脈絡、本資料夾收工程之外的學習。一個知識點若同時有工程應用（例如某演算法的歷史），以工程價值為主就放對應教材區、以故事本身為主就放這。
