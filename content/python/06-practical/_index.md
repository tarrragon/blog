# 模組六：實戰指南

本模組將前五個模組的知識整合起來，透過三個實戰案例，教你如何在 Hook 系統中進行實際開發。

## 章節列表

| 章節 | 主題 | 關鍵收穫 |
|------|------|---------|
| [6.1](new-hook/) | 如何新增一個 Hook | 完整開發流程 |
| [6.2](extend-lib/) | 如何擴展共用模組 | 模組設計與整合 |
| [6.3](new-parser/) | 如何新增語言解析器 | 繼承 ABC 實作 |

## 前置要求

建議先完成以下模組：

- 模組一：理解 Python 模組系統
- 模組二：掌握型別提示
- 模組四：了解抽象基類（對於 6.3）
- 模組五：掌握測試技巧

## 實戰流程預覽

### 新增 Hook 的完整流程

```
1. 確定 Hook 類型和觸發時機
2. 建立腳本檔案
3. 實作核心邏輯
4. 設定 Claude Code 配置
5. 撰寫測試
6. 更新文件
```

### 擴展共用模組的流程

```
1. 分析需求和現有架構
2. 設計新功能介面
3. 實作並保持向後相容
4. 更新所有使用者
5. 撰寫測試
```

### 新增解析器的流程

```
1. 繼承 BaseParser
2. 實作 parse 方法
3. 註冊到 ParserFactory
4. 撰寫測試
```

## 學習時間

每章節約 30-45 分鐘，全模組約 90-135 分鐘
